• beforCreate: 인스턴스 생성되기 전 호출
  • created: 인스턴스 생성 후 호출, 아직 마운트는 호출되기 전
  • beforeMount: 데이터가 돔에 마운팅 되기 전 호출(ServerSideRendering에서는 호출되지 않음)
  • mounted: el에 정의한 부분에 vue 인스턴스를 포함시킨 후, 즉 모든 화면이 랜더링이 완료되었을 때 호출
  • beforeUpdate: vue 인스턴스에 생성한 데이터가 변경될 때 가상돔이 재 랜더링, 패치가 되기 전 호출(SSR 호출X)
  • update: vue 인스턴스에 생성한 데이터가 변경될 떄 가상 돔이 재 랜더링, 패치가 된 후 호출
  • beforeDestroy: vue 인스턴스를 제거하기 전에 호출(SSR 호출X)
  • destroy: vue 인스턴스가 제거된 후 호출
